What is software?

-> Software is a set of programs (sequence of instructions) that allows the users to perform a well-defined function or some specified task.

What is software engineering?

-> Software engineering is defined as a process of analyzing user requirements and then designing, building, and testing software application     which will satisfy those requirements.

explain types of software?

-> types:
         1. System Software:-
         -> Operating system (OS)
         -> ex: Android, ios, windows, linux, etc..

         2. Application Software:-
         (i) Desktop Application
         -> ex: VS code, notepad, firefox, etc.. 

         (ii) Mobile Application
         -> ex: Instagram, netflix, snap, etc..

         (iii)Web Application
         -> ex: Web browser  

         3. Programming Software:-
         -> A program that can be written in any programming language to perform specific task.
         -> Compiler
         -> Interpreter

         4. Deriver Software:-
         -> ex: bluetooth deriver, audio deriver, etc..

         5. Middleware Software:-
         -> It's an intermediary software between two or more applications/software.
         -> ex: database
