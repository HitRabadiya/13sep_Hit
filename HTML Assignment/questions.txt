9. How is Cell Padding different from Cell Spacing? With example.
-> Cell Padding is used to create a border around the content area of a web page, wheras cell spacing is used for positioning elements within that content area.
   Example: <tr cellspacing="2"><td>Content</td></tr>
	    <tr cellpadding="2"><td>Content</td></tr>     

10. How can we club two or more rows or columns into a single row or column in an HTML table? With example.
->  We can club rows or columns by using rowspan or colspan.
    Example: <tr><td rowspan="3">Content</td></tr>
             <tr colspan="2"><td>Content</td></tr>
